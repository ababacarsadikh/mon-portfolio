Détection d'objets avec YOLOv8

Créer un environnement Python, préparer un dataset, entraîner un modèle YOLOv8 et tester la détection en image, vidéo ou webcam.

Retour aux tutoriels Voir Arduino
Installation Dataset Entraînement Prédiction Webcam

1. Installation

YOLOv8 est fourni par le paquet Python ultralytics. L'idéal est de travailler dans un environnement virtuel dédié.

Terminal
python -m venv .venv
.venv\Scripts\activate
pip install ultralytics opencv-python
Après installation, la commande yolo checks permet de vérifier que l'environnement est prêt.

2. Préparer le dataset

Un dataset YOLO contient les images et les annotations au format texte. Chaque ligne d'annotation décrit une classe et une boîte englobante normalisée.

Structure
dataset/
  images/
    train/
    val/
  labels/
    train/
    val/
  dataset.yaml
dataset.yaml
path: dataset
train: images/train
val: images/val

names:
  0: objet

3. Entraîner un modèle

Pour commencer rapidement, on peut utiliser yolov8n.pt, une version légère adaptée aux tests et aux machines modestes.

Commande YOLO
yolo detect train data=dataset/dataset.yaml model=yolov8n.pt epochs=50 imgsz=640
  • epochs règle le nombre de passages sur le dataset.
  • imgsz règle la taille des images utilisées pendant l'entraînement.
  • Les résultats sont enregistrés dans runs/detect/train.

4. Tester les prédictions

Une fois l'entraînement terminé, le meilleur modèle se trouve généralement dans runs/detect/train/weights/best.pt.

Image ou vidéo
yolo detect predict model=runs/detect/train/weights/best.pt source=image.jpg
yolo detect predict model=runs/detect/train/weights/best.pt source=video.mp4
Si les détections sont instables, il faut souvent améliorer les annotations ou ajouter des exemples plus variés.

5. Détection webcam en Python

OpenCV permet de lire la webcam et YOLOv8 peut annoter chaque frame en temps réel.

Python
import cv2
from ultralytics import YOLO

model = YOLO("runs/detect/train/weights/best.pt")
cap = cv2.VideoCapture(0)

while True:
    ok, frame = cap.read()
    if not ok:
        break

    results = model(frame)
    annotated = results[0].plot()
    cv2.imshow("YOLOv8", annotated)

    if cv2.waitKey(1) & 0xFF == ord("q"):
        break

cap.release()
cv2.destroyAllWindows()
